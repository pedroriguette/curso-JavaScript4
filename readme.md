**Exercicios feitos no modulo 4 de JavaScript do curso JavaScript FullStack da OniBit Code**

- Sobre o curso:

Neste módulo, exploramos os recursos modernos do JavaScript introduzidos pelas versões mais recentes do ECMAScript, como template literals, arrow functions, desestruturação, spread, 
rest e operadores avançados. Aprendeu a organizar o código com módulos (CommonJS e ES Modules) e a gerenciar dependências e scripts com o npm, utilizando também bibliotecas externas 
como o dayjs. Além disso, foi ensinado como transpilar e empacotar projetos com Babel e Webpack, dominando o fluxo de desenvolvimento moderno de aplicações JavaScript. Por meio de 
exercícios práticos e projetos estruturados, consolidamos a capacidade de escrever código mais limpo, modular e compatível com diferentes ambientes
